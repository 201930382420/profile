# 李雨帆 Sail｜个人简历网站鼠标响应版

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000`。

## 构建

```bash
npm run build
npm run start
```

## 部署到 Vercel

1. 将本目录上传到 GitHub。
2. 登录 Vercel。
3. Import Project，选择该 GitHub 仓库。
4. Framework 选择 Next.js。
5. Build Command 使用默认 `next build`。
6. 点击 Deploy。

## 本地 HTML 预览

交付包根目录中包含：

- `李雨帆_Sail_个人简历网站_鼠标响应版_本地预览.html`

可直接双击打开。
